# Statistical Learning — Applied Exercises

Implementation of statistical and machine learning methods in R, following *An Introduction to Statistical Learning* (James et al.).

**Topics covered:** Linear and logistic regression, resampling methods (cross-validation, bootstrap), regularization (Ridge, Lasso), tree-based methods (Random Forest, Boosting), SVM, unsupervised learning (PCA, clustering)

**Stack:** R, tidyverse, ggplot2, glmnet, randomForest, e1071
